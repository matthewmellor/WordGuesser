Objectives:

    1. User is able to type in a word to guess a particular word
    2. Game will show the number of letter that are incorrect in the guess
        1. Guess will still be displayed (In an old line)
    3. Game will have 3 settings
        1. Easy
            -10 guesses
        2. Medium
            -5 guesses
        3. Hard
            -3 guesses
